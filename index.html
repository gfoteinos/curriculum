A few words about the app

University curriculum is a simple curriculum management software. It allow you to create accounts for faculty members and students.

A faculty member can create courses and modules connected with courses asign courseworks and exams for a module and monitor students progress.

A student can monitor his progress and can have access to different informations like deadlines for courseworks and exams module marks e.t.c.

This project created with vanilla JavaScript & Bootstrap.
